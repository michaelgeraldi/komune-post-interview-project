Buatlah satu project website, yang dapat menyelesaikan masalah berikut: 
1. Buatlah satu halaman, dimana user dapat mengupload sebuah dokumen berformat .csv (contoh diberikan Book1 .csv).
2. Setelah di upload, maka akan tertampil di halaman website dalam bentuk tabel, data2 yang dimasukan.
3. Setelah user menakan tombol "simpan", data dapat di download dalam bentuk JSON.
4. Soal Bonus - Setelah tombol simpan ditekan, maka data akan tersimpan ke database.
5. Soal Bonus - User dapat mengedit data2 yang ada di tabel sebelum disimpan ke JSON/Database.

Hasil pengerjaan akan dinilai dengan poin berikut, berdasarkan urutan yang paling penting:
Keselesaian tugas. (Apakah sesuai dengan instruksi?)
Kerapihan kode 
User experience 
Kerapihan tampilan
